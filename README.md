# Hepsiburada-Automation
1. Kullanıcı girişi yapılarak sepete ürün eklenmesi
Kullanıcı Hepsiburada.com sitesini ziyaret eder.
Kullanıcı giriş işlemi yapılır.
Yönlendirmeden sonra anasayfada kullanıcı giriş işleminin yapıldığı doğrulanır
Kullanıcı, nutella ürün için arama yapar.
Kullanıcı, Arama sonucunda ekrana gelen ürün listesden ürünü seçer.
Seçilen ürün için 2 tane farklı satıcıdan ürün seçilip sepete eklenir.
Seçilen ürünün doğru olarak eklendiği ‘Sepetim’ sayfasında doğrulanır.

2. Kullanıcı girişi yapılmadan belirtilen ürünü sepete ekleme
Kullanıcı Hepsiburada.com sitesini ziyaret eder.
Kullanıcı, nutella ürün için arama yapar.
Seçilen ürün için 2 tane farklı satıcıdan ürün seçilip sepete eklenir.
Seçilen ürünün doğru olarak eklendiği ‘Sepetim’ sayfasında doğrulanır.
